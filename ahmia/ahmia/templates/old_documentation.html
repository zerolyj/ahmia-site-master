Security measure: Obfuscate real-time stats. The clicks are rounded to the next multiple of 8. This simple way prevents leaking too accurate real-time statistics.

Examples:

  • TOP 5 ordered by Tor2web visit count:
    • https://ahmia.fi/stats/popularity?limit=5&offset=0&order_by=tor2web
  • The 10th from ordered by public WWW backlinks:
    • https://ahmia.fi/stats/popularity?limit=9&offset=8&order_by=public_backlinks
  • The first from ordered by number of search result clicks:
    • https://ahmia.fi/stats/popularity?limit=1&offset=0&order_by=clicks

Onion information

All banned hidden services: https://ahmia.fi/banned/
These are MD5 sums calculated from something.onion\c part (echo -e "BLAHBLAHBLAH.onion\c" | md5sum).

All known non-banned onion domains: https://ahmia.fi/onions/

See if a hidden service has been online (Down/Up) within a week: https://ahmia.fi/address/jntlesnev5o7zysa/status

Ask a hidden service's information:
curl -i --insecure -H "Content-Type: application/json" https://ahmia.fi/address/jntlesnev5o7zysa/
curl -i --insecure -H "Content-Type: application/rdf+xml" https://ahmia.fi/address/jntlesnev5o7zysa/

Ask all hidden services' information:
curl -i --insecure -H "Content-Type: application/json" https://ahmia.fi/address/
curl -i --insecure -H "Content-Type: application/rdf+xml" https://ahmia.fi/address/
